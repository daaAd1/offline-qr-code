相比于其它使用 Google Web APIs 来生成二维码的扩展，此扩展完全离线工作。此二维码生成器将你的隐私放在在首位！🔐 它通过离线工作来实现该目标, 完全不依赖于网络！ 👌 简单易用，点击一次就能使用！👌 它拥有极端简洁，但功能强大的用户界面，你可以在设置中调整多个详细设置。尽管如此，它还是非常容易使用！比如，你可以拖放鼠标调整二维码大小。轻量化所以体积小，速度快，安装简单，甚至使用手机流量安装都没压力。 📦 可生成多种二维码！ 📦 可生成彩色二维码, 保存二维码或只用它们来从桌面发送标签页或文本到你的手机上！由你选择！ 你的二维码只需要点击一次然后你就能即时输入并改变二维码中的文本内容！ 📢 更多功能 📢
  • 将你的隐私放在在首位！在这里隐私是默认项，所以它离线生成二维码。
  • 遵循 Firefox Photon 设计模式，无缝集成到你的 Firefox 中。
  • 非常简洁，易用！
  • 使用最新的，强大的，可定制的 QR code libraries 来生成二维码.
  • 你可以生成并保存二维码为 SVG 或 PNG 图像 (Canvas)！
  • 你可以配置大小，颜色和二维码的其它样式。
  • 使用快捷键(Ctrl+Shift+F10)来生成二维码。
  • 在网站上选择文字生成二维码。
  • 完整支持Unicode/UTF-8/Emoji。
  • 在桌面版和移动版上观感很好, 比如，容易控制！
  • 已翻译为多种语言。你可以在 贡献你的语言翻译！
  • 兼容 Android 版 Firefox。
  • 设置可在多设备间同步。
  • 设置可由管理员控制。(正在实现)
